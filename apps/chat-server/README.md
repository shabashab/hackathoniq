# Setup

## Pre-requisites

You should have `air` and `swag` installed:

```
go install github.com/air-verse/air@latest
go install github.com/swaggo/swag/cmd/swag@latest
```

## Running

```
air
```

# Toolkit

- DI: https://github.com/uber-go/fx
- Logger: https://github.com/uber-go/zap
